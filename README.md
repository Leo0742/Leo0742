## Featured projects

### [Lost & Found Board](https://github.com/Leo0742/Lost-Found-Board)

My strongest backend portfolio project: a full-stack lost-and-found platform with a **FastAPI** backend, **PostgreSQL** database, **React** frontend, Telegram bot, report lifecycle, search/matching, claims, moderation workflow, health checks, and Docker Compose deployment.

`Python` · `FastAPI` · `SQLAlchemy` · `PostgreSQL` · `React` · `Telegram Bot` · `Docker`

### [Inno Code](https://github.com/Leo0742/Inno-Code)

A desktop coding assistant that runs multi-agent planning and keeps repository changes behind explicit review and approval. The Electron app supports persisted sessions, exact sandbox previews, selective patch application, runtime logs, provider profiles, and validation commands.

`TypeScript` · `Electron` · `React` · `Node.js` · `Git` · `Developer Tools`

### [SmartRoute VPN](https://github.com/Leo0742/SmartRoute-VPN_Site)

A full-stack subscription dashboard with a landing page, authentication, user cabinet, admin panel, support tickets, partner workflows, Telegram login, Prisma persistence, and Docker deployment.

`JavaScript` · `React` · `Express` · `Prisma` · `SQLite` · `Docker`

### [QuizMind Platform](https://github.com/Leo0742/QuizMind_Platform) <sub>contributed project · fork</sub>

A contributed fork of a SaaS-style platform foundation with a Next.js web app, NestJS API, worker service, shared packages, PostgreSQL, Redis, Docker, auth, feature flags, remote config, and support/admin workflows.

`TypeScript` · `Next.js` · `NestJS` · `Prisma` · `PostgreSQL` · `Redis` · `Docker`

### [CRYPTOGRAPHER](https://github.com/Leo0742/CRYPTOGRAPHER)

A Python/PyQt desktop utility for protecting local files through a graphical interface and packaging the application for Windows users.

`Python` · `PyQt5` · `Desktop` · `Packaging`
