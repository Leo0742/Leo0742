## Featured projects

### [Lost & Found Board](https://github.com/Leo0742/Lost-Found-Board)

A full-stack lost-and-found system with a **FastAPI** backend, **React** frontend, **PostgreSQL** database, and Telegram bot. It includes report management, search and matching, claim workflows, moderation, tests, and Docker-based deployment.

`Python` · `FastAPI` · `React` · `TypeScript` · `PostgreSQL` · `Docker`

### [Inno Code](https://github.com/Leo0742/Inno-Code)

An alpha-stage desktop coding assistant that runs multi-agent planning and places repository changes behind explicit review and approval. The Electron app supports persisted sessions, exact sandbox previews, selective patch application, runtime logs, and validation commands.

`TypeScript` · `Electron` · `React` · `Node.js` · `Git`

### [QuizMind Platform](https://github.com/Leo0742/quizmind-platform) <sub>contributed project · fork</sub>

A contributed fork of a full-stack platform foundation with a Next.js web app, NestJS API, background workers, shared domain packages, PostgreSQL, Redis, and Docker deployment. The repository covers auth, billing, RBAC, feature flags, remote configuration, and support/admin workflows.

`TypeScript` · `Next.js` · `NestJS` · `Prisma` · `PostgreSQL` · `Redis`

### [RDevents](https://github.com/Leo0742/RDevents) <sub>contributed project · fork</sub>

A contributed fork of an event-management platform with public event discovery, registration, team and volunteer flows, administration, analytics, and observability. It is organized as a TypeScript monorepo with a Next.js frontend and Express/Prisma backend.

`TypeScript` · `Next.js` · `Express` · `Prisma` · `PostgreSQL` · `Docker`

### [CRYPTOGRAPHER](https://github.com/Leo0742/CRYPTOGRAPHER)

A Python/PyQt desktop utility for encrypting and decrypting binary files. The project implements AES, DES, and RSA workflows and packages the application for Windows users.

`Python` · `PyQt5` · `Desktop` · `Cryptography`
